安装教程

下载并解压 点击“下载 Windows 版”，把 ZIP 解压到任意文件夹。
启动本地服务 双击运行“YOUTUBE下载器.bat”，保持服务窗口不要关闭。
安装 Chrome 扩展 打开 Chrome 扩展管理页面，开启“开发者模式”，点击“加载已解压的扩展程序”，选择 extension 文件夹。
粘贴链接下载 复制视频链接或分享文本，点击浏览器右上角“媒体下载器”，粘贴后点击“开始下载”。

常见问题

提示本地服务未启动怎么办？

先双击运行解压目录里的 YOUTUBE下载器.bat，再重新打开扩展。

抖音、B 站需要登录吗？

部分内容需要。请先在 Chrome 里登录对应平台，并确认网页能正常播放。

为什么下载速度慢？

速度取决于平台 CDN、当前网络和视频大小。工具已启用重试和分块下载。
