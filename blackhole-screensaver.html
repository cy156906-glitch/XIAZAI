桌面黑洞屏保

基于 Eric Bruneton 黑洞着色器的 Windows 桌面黑洞可视化程序。空闲自动显示，看视频时自动隐藏，14 个参数可调。

Windows 10/11 OpenGL + D3D11 空闲检测 完全本地运行 MIT License
下载 Windows 版 查看源码 / GitHub
本软件仅在你的本机运行，不联网，不上传任何数据。OpenGL 渲染可能需要较新的显卡驱动。

是什么

在桌面上渲染一个真实的史瓦西黑洞——引力透镜、吸积盘、光子环、多普勒效应、相对论性光束都按物理模型实时计算。空闲时显示，动一下鼠标就消失，不会打扰你工作。

主要功能

空闲自动触发

设置空闲 N 秒后显示，默认 5 分钟。动鼠标/键盘立即消失。

看视频自动隐藏

检测到浏览器、游戏、视频播放器在前台时，黑洞不打扰你。

14 个可调参数

色温、倾角、旋转、半径、不透明度、多普勒、光束、亮度、对比、缠绕、转速、曝光、星空亮度。

预设管理

支持保存/复制/粘贴预设，多预设顺序/循环/随机播放。

托盘退出

右下角托盘图标右键可随时退出，资源占用低。

完全本地

不联网，不上传数据，所有渲染在你自己的电脑上完成。

两种工作模式

模式行为
始终显示黑洞常驻桌面
空闲检测空闲 N 秒后显示，活跃时自动隐藏

使用方法

  1. 下载并解压 ZIP 到任意文件夹。
  2. 双击 release\blackhole.exe（首次运行会弹出配置窗口）。
  3. 调整参数 → 点击"启动"。
  4. 黑洞会在空闲时自动出现在桌面上，动鼠标/键盘即消失。
  5. 右下角托盘图标 → 右键菜单 → 退出。
如果双击 exe 没反应：需要 Windows 10/11，并安装较新的显卡驱动（OpenGL 4.3+）。如果使用 RDP 远程桌面，黑洞渲染可能不可见。

常见问题

需要联网吗？

不需要。所有计算都在本地完成，关闭网络也能正常使用。

占用 CPU/GPU 高吗？

黑洞空闲时渲染开销很低（约 5-10% GPU），只在屏幕显示时工作。

支持多显示器吗？

支持，会在所有显示器上同时显示黑洞效果。

怎样彻底关闭？

右下角托盘图标 → 右键 → 退出。或在配置页面取消勾选"启动"。
