安装方法（约 2 分钟）

  1. 下载并解压工具包到任意位置（例如桌面）。
  2. 进入解压后的文件夹，双击 安装开机自启.bat。这个脚本会做两件事：把本地服务注册为 Windows 开机自启，并且立刻启动它（无黑窗、后台运行）。只需要这一次，以后每次开机服务都会自动起来。
  3. 在 Chrome 地址栏输入 chrome://extensions/，打开右上角"开发者模式"。
  4. 点击"加载已解压的扩展程序"，选择解压文件夹里的 extension 文件夹。
  5. 点击浏览器右上角拼图图标，把"媒体下载器"固定到工具栏，方便以后用。

使用流程

  1. 在 Chrome 里正常登录你要下载的平台（抖音、B站、小红书、YouTube 等），播放任意视频让登录状态生效。
  2. 复制视频页面链接或分享文本。
  3. 点击工具栏的"媒体下载器"扩展图标，把链接粘进去，点击"下载"。
  4. 文件默认保存到桌面的 MediaDownloads 文件夹。下载完成后点绿色的"已下载到"提示可直接打开。

常见问题

扩展提示"本地服务未启动，请先双击运行 YOUTUBE下载器.bat"
说明服务没在跑。双击解压文件夹里的 YOUTUBE下载器.bat（无黑窗，瞬间就关），服务会在后台启动。常见原因：注册表自启被杀毒软件清掉、电脑长时间没重启、或者第 2 步跳过了。

下载某个平台失败
先确认 Chrome 里已经登录该平台并能正常播放该视频。再不行，把链接直接粘到扩展里重试一次。

抖音提示 Cookie 或登录错误
打开 douyin.com，播放任意视频几秒钟让 Cookie 生效，然后回扩展重试。

包内文件说明

  • 安装开机自启.bat — 双击一次，注册 Windows 开机自启并启动服务。
  • YOUTUBE下载器.bat — 手动启动服务（备用）。
  • 卸载开机自启.bat — 取消开机自启并停止服务。
  • extension/ — Chrome 扩展源文件。
  • python/ — 内置 Python 3.11 + 13 个预装依赖，免装 Python。
  • yt-dlp.exe、ffmpeg.exe — 下载与转码工具。
  • server.py — 本地下载服务主程序。
  • 使用说明.txt — 离线版完整说明。

提示：建议把整个解压文件夹放在一个固定位置（例如 D:\Tools\MediaDownloader），不要放在桌面或下载文件夹里，避免被清理工具误删。
