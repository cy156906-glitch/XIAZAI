## 安装

1. 下载并解压 ZIP。
2. 双击运行 `YOUTUBE下载器.bat`。
3. 打开 Chrome 扩展管理页面。
4. 开启“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压目录里的 `extension` 文件夹。
7. 复制视频链接，打开扩展，粘贴后点击“开始下载”。
